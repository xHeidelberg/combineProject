# 💊 Pharmacy Management System

## 📌 Overview
The **Pharmacy Management System Dashboard** is a responsive web-based interface designed to manage pharmacy operations efficiently. This project serves as a **partial fulfillment of school requirements**, showcasing frontend development using modern UI/UX practices.

The dashboard provides role-based navigation and features for managing pharmacy activities such as products, orders, appointments, and more.

---

## 🎯 Objectives
- To develop a modern and responsive dashboard UI  
- To implement role-based navigation **(Admin, Pharmacist, Staff, Patient)**  
- To demonstrate frontend integration using HTML, TailwindCSS, and JavaScript  
- To simulate a real-world pharmacy management system interface  

---

## 🛠️ Technologies Used
- **HTML5**
- **TailwindCSS / CDN**
- **JavaScript (Vanilla JS)**
- **PHP**
- **Google Fonts (DM Sans, DM Mono)**

---

## ✨ Features
- 📊 Interactive Dashboard UI  
- 🌙 Dark Mode Toggle (saved in localStorage)  
- 📱 Fully Responsive Layout  
- 📂 Sidebar Navigation (collapsible)  
- 🔐 Role-Based Menu Visibility  
- 🔔 Notification UI  
- 🔍 Search Bar (UI only)  
- 🧾 Modular Navigation (Products, Orders, Patients, etc.)  

---

## 📚 Academic Context

This project is developed as part of academic coursework to demonstrate:
